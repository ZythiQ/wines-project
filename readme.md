# Wine Space

> An interactive 3D visualization of personal wine tasting notes.
> Site at: [zythiq.github.io/wines-project/](https://zythiq.github.io/wines-project/)

## Overview

This project maps 70 wines tasted in and outside class into a navigable 3D scatter plot. Each wine is encoded across 22 categories, then projected into three dimensions so wines that *taste alike* end up *near each other.*

> One important note is that there is no rhyme or reason to the 3 axis you can navigate as these represent several dimensions each.
